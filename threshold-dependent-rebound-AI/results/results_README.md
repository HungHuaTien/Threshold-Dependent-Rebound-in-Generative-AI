## Files

### Tables and Reports

#### `B1_Newey_West_Results.txt`
**Source**: `code/run_B1_B2_analysis.py`  
**Content**: Newey-West robust standard errors for Chow Test

**Key findings**:
- DeepSeek-R1: 5.3σ (95% CI: [3.8σ, 6.7σ]), t=4.34, p<0.001
- GPT-5: 3.0σ (95% CI: [1.5σ, 4.4σ]), t=4.67, p<0.001

---

#### `B2_Control_Group_Results.csv`
**Source**: `code/run_B2_Control_Group_analysis.py`  
**Content**: Detailed Chow Test results for AI packages vs control group

**Columns**:
- `package`: Package name
- `event`: Event name (DeepSeek-R1 or GPT-5)
- `sigma_jump`: σ jump magnitude
- `f_stat`: F-statistic
- `p_value`: p-value
- `type`: Package type (AI套件 or 對照組)

**Sample**:
```csv
package,event,sigma_jump,f_stat,p_value,type
anthropic,DeepSeek-R1,3.69,45.20,0.0001,AI套件
numpy,DeepSeek-R1,1.05,0.53,0.5925,對照組
```

**Source**: `code/run_B2_Control_Group_analysis.py`

## Interpreting Results

### Significance Levels

In all tables:
- `***` : p < 0.001 (highly significant)
- `**`  : p < 0.01 (very significant)
- `*`   : p < 0.05 (significant)
- (blank) : p > 0.05 (not significant)

### σ Jump Interpretation

**Magnitude classification**:
- σ > 10 : Extreme jump (rare event)
- 3 < σ < 10 : Large jump (structural break)
- 1.96 < σ < 3 : Moderate jump (95% threshold)
- σ < 1.96 : Normal volatility

**For AI packages** (DeepSeek-R1 event):
- anthropic: 3.69σ → Large structural break
- openai: 2.41σ → Moderate jump
- langchain: 2.13σ → Moderate jump

**For control group**:
- All < 2σ → Normal volatility range
